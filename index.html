Multi-Source RAG System

Developed a Multi-Source Retrieval-Augmented Generation (RAG) system to extract and query insights from PDFs, websites, and YouTube videos. Integrated LangChain and ChromaDB for efficient document embedding and vector-based retrieval, and deployed local LLMs using Ollama to enable offline, cost-effective summarization and question answering. Built using Python and advanced NLP techniques.

Hindi Handwritten Swar Recognition Using Neural Networks from Scratch

Developed a 2-layer Neural Network from scratch in C++ without using external ML libraries. Implemented manual forward and backward propagation algorithms to optimize model weights and accuracy. Constructed a noise-augmented dataset to improve model generalization for Hindi vowel classification.

View

Hand Gesture Recognition for Presentation Control

Developed an interactive system using hand gesture recognition to control PowerPoint slides. Utilized cvzone and cv2 for real-time hand tracking and gesture detection, enabling actions such as slide navigation and annotation drawing based on specific hand gestures.
